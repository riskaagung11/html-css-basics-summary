HTML & CSS Basics Summary

HTML Summary

HTML (HyperText Markup Language) is used to define our page content, structure and meaning. you don't use it for styling purposes. Use CSS for that instead!

  1. HTML uses "elements" to describe (annotate) content
  2. HTML element typically have an opening tag, content and closing tag
  3. You can also havve void (empty) elements like images
  4. you can also configure elements with attributes
  5. there's a long list of available elements but you'll gain exprience over time, no worries.

Learn more about all available HTML elements on the MDN HTML element reference

CSS Summary

CSS (Cascading Style Sheets) is used for styling your page content.

  1. Style are assigned via property-value pairs
  2. You can assign style via the "style" attribute
  3. To avoid code duplication, you typically use global style(e.g. via the "style" element) instead
  4. Alternatively, you can work with external stylesheet files which you "link" to
  5. When Working with CSS, conscepts like "inheritance","specificity" and the "box model" should be understood.

Learn more about all available CSS properties and values on The MDN CSS property reference
